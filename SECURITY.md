## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability within this project, please send an e-mail to security@example.com. All security vulnerabilities will be promptly addressed.

Please do NOT open an issue for a security vulnerability, as it makes the vulnerability public before we can provide a fix.

### Disclosure Policy

- We will acknowledge receipt of your report within 48 hours.
- We will provide a fix within 7 days for critical vulnerabilities.
- We will coordinate public disclosure after the fix is released.
