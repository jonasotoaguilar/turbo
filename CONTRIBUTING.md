# Contributing to Turbo

First off, thank you for considering contributing to Turbo! It's people like you that make the open-source community such an amazing place.

### Reporting Bugs

- Check the [Issues](https://github.com/jonasotoaguilar/turbo/issues) to see if it has already been reported.
- Use a clear and descriptive title.
- Describe the exact steps which reproduce the problem.

### Suggesting Enhancements

- Check the Issues to see if the idea has already been suggested.
- Explain why this enhancement would be useful to most users.

### Your First Code Contribution

- Fork the repository.
- Clone the fork to your local machine.
- Create a new branch: `git checkout -b my-new-feature`
- Make your changes and commit them following [Conventional Commits](https://www.conventionalcommits.org/).
- Push your branch: `git push origin my-new-feature`
- Submit a Pull Request.
